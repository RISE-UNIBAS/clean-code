The folder [`clean-code/git`](https://github.com/RISE-UNIBAS/clean-code/tree/main/git) contains:

- slides as HTML in [`clean-code/git/git.html`](https://github.com/RISE-UNIBAS/clean-code/tree/main/git/git.html)
